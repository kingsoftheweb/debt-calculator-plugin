Debt Calculator Plugin

Documentation

Description

This plugin allows you to hook into the ARMember plugin, and adds the debt calculator to it.

Shortcodes

Debt Calculator Shortcode

Default code example:

debt-calculator

- This shortcode shows the debt calculator wherever it is used.
