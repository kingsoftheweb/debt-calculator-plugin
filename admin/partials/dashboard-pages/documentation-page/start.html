Debt Calculator Plugin

Documentation

Description

This plugin allows you to hook into the ARMember plugin, and adds the debt calculator to it.

Sections

Debts Reports

All debts for the current user are listed here, with two buttons for export ( PDF and Excel).

Add New Debt

A user or an admin can add a new debt for the current user from this section, by filling the data of the debt and press on "Add new Debt".

Update A Debt

A user or an admin can add new Payment for a specific debt for the current user from this section, by inserting the amount of the payment and press on "update debt".

The value will be registered at the Current Time and Date of once you press on update debt.

Debts

All debts for the current user are listed here, with more specific info per each debt, that you can retrieve when you press on the name of the debt.

Shortcodes

Shortcode are an easy way that worpdress offers to show and execute specific code on specific parts of your website exclusively. Read more about shortcodes here.

Debt Calculator Shortcode

Shortcode to be used in any Post or Page on your website:

[debt-calculator]

Objective

This shortcode will show append the debt calculator dashboard to any page or post, it will be tied to the current visitor of the page/post.

ARMember

In your webiste, after you activate the ARMemeber plugin, and activate the profile page. The plugin will automatically hook into the profile page, and add the calculator debt dashaboard for each memeber.

To activate profile module in the ARMember plugin:
  • Go to : Armember>Modules Activate "Social Feature" module.
  • Create a new page, and call it "Profile" or "My Profile". and make sure to insert the proper shortcode for the Profile Page, it usually starts with : 'arm_account_detail'
  • Publish the page, and the debt calcualtor dashbaord will be appened automatically to the bottom of the profile page.

Exporting

This plugin allows two types of exports.

  • 1- PDF Export
  • 1- Excel Export

PDF Reports Export

Any member can export their payments/debts info into a 4-page pdf report, that is ready to print into a true color A4 paper with scalable options.
Data of Payments is for the last month.

Excel Reports Export

Only Admins can export reports to excel, which includes all payments for the specific user.
